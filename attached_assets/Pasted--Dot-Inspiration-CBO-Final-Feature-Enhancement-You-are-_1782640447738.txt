# Dot Inspiration CBO - Final Feature Enhancement

You are working on an existing production website.

This is NOT a redesign.

Do NOT rebuild the application.

Do NOT change the technology stack.

Do NOT break existing routing or functionality.

Extend the existing project while preserving all current features.

## PRIMARY OBJECTIVE

Transform the website into a professional NGO platform where nearly all website content can be managed through the Admin Portal instead of editing code.

Everything added should integrate with the existing Admin Portal.

---

# 1. LEADERSHIP MANAGEMENT

Create a complete Leadership Management module.

Public page:

/leadership

The page should display professional leadership profiles.

Each profile includes:

* Photo
* Full name
* Position
* Biography
* Qualifications
* Department
* Email (optional)
* Phone (optional)
* LinkedIn (optional)
* Display order
* Active/Inactive status

Clicking a profile opens a detailed page.

Inside Admin Portal create:

Leadership Management

Administrator can:

* Add leader
* Edit leader
* Delete leader
* Upload photo
* Reorder leaders
* Hide/show leaders
* Feature leaders

Changes should appear immediately on the website without editing code.
